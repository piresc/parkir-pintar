SERVICES Microservices Architecture

  • Gateway — HTTP :8082 | Gin framework | REST API gateway, auth middleware, rate limiting
  • Reservation — gRPC :9091 | Core domain aggregate, lifecycle state machine, orchestrates billing & payment
  • Search — gRPC :9092 | CQRS read model, spot availability queries, eventually consistent via NATS
  • Billing — gRPC :9093 | Fee calculation engine, invoice generation, pricing rules
  • Payment — gRPC :9094 | QRIS/card/e-wallet processing, stub payment gateway

EVENTS Event-Driven Architecture

  • reservation.search.spot-updated — Notifies Search service when spot availability changes
  • reservation.analytics.* — Analytics events for reporting and dashboards
  • payment.reservation.paid — Payment confirmation triggers reservation state transition
  • payment.reservation.failed — Payment failure triggers cancellation flow
